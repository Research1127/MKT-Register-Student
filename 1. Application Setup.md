# Tuition Fee Access Portal (MVP)

## Overview

The **Tuition Fee Access Portal** is a simple web application that allows parents to check their **tuition payment status** before accessing a Google Form.

Parents must enter their **IC number** to verify whether they have any outstanding tuition fees. Only parents with **fully paid fees** can access the Google Form.

This project is designed as a **Minimum Viable Product (MVP)** and uses free tools such as **Google Sheets, Google Apps Script, and GitHub Pages**.

---

# System Architecture

```
Parent (Browser)
       │
       ▼
GitHub Pages (Frontend - HTML, TailwindCSS, JavaScript)
       │
       ▼
Google Apps Script API
       │
       ▼
Google Sheet (Database)
```

## Flow

1. Parent enters **IC number**
2. Website sends request to **Google Apps Script API**
3. Apps Script checks **Google Sheet database**
4. System returns payment status

Possible responses:

- **Dibayar** → Parent can access Google Form
- **Tertunggak** → Warning message shown
- **IC not found** → Error message displayed

# Setup Instructions

## 1. Prepare Google Sheet

Create a Google Sheet with the following structure:

| IC Number      | Name | Status     |
| -------------- | ---- | ---------- |
| 900101-01-1234 | Ali  | Dibayar    |
| 880202-02-5678 | Siti | Tertunggak |

Column descriptions:

- **IC Number** → Parent identification number
- **Name** → Parent or student name
- **Status** → Payment status (`Dibayar` or `Tertunggak`)

---

# 2. Create Google Apps Script API

Open the Google Sheet.

Navigate to:

```
Extensions → Apps Script
```

Add the following script:

```javascript
function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Sheet1");
  const data = sheet.getDataRange().getValues();

  const ic = e.parameter.ic;

  for (let i = 1; i < data.length; i++) {
    if (data[i][0] == ic) {
      return ContentService.createTextOutput(
        JSON.stringify({
          status: data[i][2],
        }),
      ).setMimeType(ContentService.MimeType.JSON);
    }
  }

  return ContentService.createTextOutput(
    JSON.stringify({
      status: "NotFound",
    }),
  ).setMimeType(ContentService.MimeType.JSON);
}
```

# 4. Configure Frontend

In `index.html`, update the API URL:

```javascript
const url = "YOUR_SCRIPT_URL?ic=" + ic;
```

Example:

```
https://script.google.com/macros/s/XXXX/exec?ic=900101-01-1234
```

Also update the **Google Form URL**.

### Full Code

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Tuition Access Portal</title>
    <script src="https://cdn.tailwindcss.com"></script>
  </head>

  <body class="bg-gray-100 min-h-screen flex items-center justify-center">
    <div class="bg-white shadow-lg rounded-2xl p-8 w-full max-w-md">
      <h2 class="text-2xl font-bold text-center text-gray-800 mb-6">
        Tuition Access Portal
      </h2>

      <p class="text-sm text-gray-500 text-center mb-6">
        Sila masukkan nombor IC untuk menyemak status bayaran.
      </p>

      <div class="space-y-4">
        <input
          type="text"
          id="icNumber"
          placeholder="Contoh: 900101011234"
          class="w-full border rounded-lg px-4 py-2 focus:outline-none focus:ring-2 focus:ring-blue-500"
        />

        <button
          onclick="checkPayment()"
          class="w-full bg-blue-600 text-white py-2 rounded-lg font-semibold hover:bg-blue-700 transition"
        >
          Semak Status
        </button>
      </div>

      <div id="message" class="mt-6 text-center text-sm"></div>
    </div>

    <script>
      async function checkPayment() {
        const ic = document.getElementById("icNumber").value;
        const msg = document.getElementById("message");

        msg.innerHTML = "";

        if (!ic) {
          msg.innerHTML =
            '<p class="text-red-500 font-medium">Sila masukkan nombor IC</p>';
          return;
        }

        const url = "need to put the url here for Google Apps Script API " + ic;

        try {
          const response = await fetch(url);
          const data = await response.json();

          if (data.status === "Dibayar") {
            msg.innerHTML = `
              <p class="text-green-600 font-semibold mb-3">
                Bayaran telah dijelaskan
              </p>
              <a 
                href="https://docs.google.com/forms/d/e/1FAIpQLSeFXvfvjXZJXGDrz4aTYOG4-Whd44Ysh2uGb0BhofzZvEGDMg/viewform?usp=publish-editor"
                target="_blank"
                class="inline-block bg-green-600 text-white px-4 py-2 rounded-lg hover:bg-green-700 transition"
              >
                Buka Google Form
              </a>
            `;
          } else if (data.status === "Tertunggak") {
            msg.innerHTML =
              '<p class="text-red-500 font-medium">Sila jelaskan tunggakan bayaran dahulu</p>';
          } else {
            msg.innerHTML =
              '<p class="text-red-500 font-medium">IC tidak dijumpai</p>';
          }
        } catch (error) {
          msg.innerHTML =
            '<p class="text-red-500 font-medium">Ralat semasa menyemak data</p>';
        }
      }
    </script>
  </body>
</html>
```
